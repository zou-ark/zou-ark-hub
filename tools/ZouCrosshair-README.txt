Zou走走 准星 - 使用说明

1. 双击“ZouCrosshair.exe”。
2. 在面板中选择目标显示器、形状、颜色和尺寸。
3. 游戏画面模式请选择“无边框全屏”或“窗口化全屏”。
4. 准星默认位于所选显示器正中心，可用水平/垂直偏移微调。
5. 点击“一键恢复屏幕中心”即可归零偏移。
6. 左侧“访问资源网站”可打开 Zou 方舟资源中心，“复制链接”可复制网址。
7. 点击“联系作者 · VX：Hiizou”可复制微信号。

说明：
- 准星覆盖层不会拦截鼠标点击。
- 所有设置会自动保存。
- 点击右上角 X 只会缩小到右下角托盘，不会退出。
- 如需彻底退出，请右键托盘图标并选择“退出”。
- 联系方式：VX Hiizou。
- 本工具不注入、不读取、不修改任何游戏进程。
